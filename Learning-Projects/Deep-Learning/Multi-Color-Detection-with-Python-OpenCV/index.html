Multi-Color Detection using OpenCV

A computer vision project that detects multiple colors in real-time from a live webcam feed using Python and OpenCV. Detected colors are highlighted with bounding boxes and labeled for clarity.

Key Features

  • Uses HSV color space for accurate color detection
  • Detects multiple colors (Red, Green, Blue, Yellow, Purple, etc.)
  • Bounding boxes are drawn in the detected color itself
  • Labels displayed above each detected region
  • Simple and effective real-time detection

Use Cases

  • Traffic light detection for self-driving systems
  • Sorting objects by color in industries
  • Robotics vision tasks
  • Educational and learning projects
